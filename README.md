# 🧀 Chezz uwu

H-hi there! Welcome to **Chezz**, a tiny chess engine I built because I had some free time and wanted to see if I could make Python do a big brain move. (〃￣ω￣ゞ)

## ✨ What's inside?

- **OOP Power:** Every piece is its own little object with its own feelings and rules.  
- **Smart Pawns:** They can jump two squares at the start, but they won't jump over their friends! They also grow up to be big strong Queens when they reach the end. 👑  
- **Sliding & Stepping:** Rooks, Bishops, and Queens slide around, while Knights and Kings just take tiny (or curvy) steps.  
- **King Capture:** The game ends when the King gets eated. Nom.

## 🚀 How to Pway

1. **Clone the repo** (p-please):  
   ```bash
   git clone https://github.com/Renivier/Chezz.git
   ```

2. **Run the engine:**

   ```bash
   python main.py
   ```

3. **Input moves:** Just type `row,col` for the start and end (like `1,0` to `2,0`). Don't forget the comma!
